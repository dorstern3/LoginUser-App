- **Front &amp; Back:** Flutter

- **DB:** Firebase

## View:
- ### Android device:

## Description:
User application with the features:
- Registration and login with username and password.
- Recover password with email.
- Login with Facebook.
- Login with Google.
- Displaying the username after login (on the home page).
